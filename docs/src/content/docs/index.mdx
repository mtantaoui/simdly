## Quick Example

See Simdly in action with a simple vector addition:

<Tabs>
<TabItem label="Scalar (Slow)">

```rust
fn add_vectors(a: &[f32], b: &[f32]) -> Vec<f32> {
    a.iter().zip(b.iter()).map(|(x, y)| x + y).collect()
}
```

</TabItem>
<TabItem label="SIMD (Fast)">

```rust
use simdly::SimdAdd;

fn add_vectors_simd(a: &[f32], b: &[f32]) -> Vec<f32> {
    // Use the high-level SimdAdd trait for easy vector addition
    a.simd_add(b).unwrap_or_else(|_| {
        // Fallback to scalar addition if SIMD fails
        a.iter().zip(b.iter()).map(|(x, y)| x + y).collect()
    })
}
```

</TabItem>
</Tabs>


## Key Features

- ⚡ **Multi-Platform SIMD**: AVX2 instructions on x86/x86_64 and NEON on ARM for optimal performance across architectures
- 🔧 **Zero Configuration**: Pre-configured with `target-cpu=native` - just `cargo add simdly` and start coding
- 🚀 **Zero-Cost**: Compile-time optimizations with no runtime overhead
- 🛡️ **Memory Safe**: Rust's safety guarantees with optimized unsafe internals
- 🔧 **Universal API**: Write once, run optimally on both Intel/AMD and ARM processors
- 📊 **Benchmarked**: Thoroughly tested performance characteristics across platforms
- 🎯 **Production Ready**: Used in high-performance computing applications on diverse hardware
